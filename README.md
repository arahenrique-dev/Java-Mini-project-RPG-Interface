# RPG Interface Swing

## Pour lancer le programme

Dans le dossier dans le terminal: 

    javac GameGUI.java 
    java GameGUI


## Comment jouer
Tout d'abbord créez votre personnage.
Vous pouvez vous déplacer avec W, A, S, D. 
Pour ouvrir votre inventaire, appuyez sur Q.
Le but est d'arriver tout en bas et à la droite de la carte.
